# 📚 LibraFlow — University Library Management System

## Overview
LibraFlow is a web-based system that manages book borrowing, 
returns, fines, and notifications for university students and staff.

## Features
- 🔍 Book search and catalog management
- 📖 Borrow & return tracking
- 💰 Automated fine calculation
- 🔔 Due date reminders & notifications
- 📊 Admin reports and analytics

## Tech Stack
- Backend: FastAPI (Python) / Node.js
- Frontend: React
- Database: PostgreSQL
- Auth: JWT

## Design Patterns Used
- Repository Pattern
- Factory Pattern
- Strategy Pattern
- Observer Pattern
